***Vue.JS 2 Install***
```
vue create frontend
```

***Install vuetify***
```
vue add vuetify
```

***Remove eslint***
```
npm remove @vue/cli-plugin-eslint
```

***Init***
```
npm init -y
```

***express Install***
```
npm install express --save
```

***Runing server.js***
```
node server.js
```

***localhost***
```
http://localhost:3000
```

npm install -g nodemon

npm install axios --save
